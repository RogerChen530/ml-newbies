「新手村逃脫！初心者的 Python 機器學習攻略」是一本寫給對於機器學習有興趣、並想使用 Python 程式語言實作的入門書。內容是改寫我在 2017 年 iT 邦幫忙鐵人賽：「R 語言使用者的 Python 學習筆記」的系列文章而成，主要針對能撰寫 Python 程式、對於機器學習理論沒有基礎卻有興趣的讀者，透過本書能夠按圖索驥，走出新手村，成功一轉！
